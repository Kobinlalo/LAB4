README – Lab 4 Konsumera web-API
================================

Beskrivning:
Applikationen hämtar JSON-data från GitHubs API och visar information om .NET Foundations publika repos i konsolen.

Hur man kör:
1. Öppna projektet i Visual Studio
2. Kör programmet med F5
3. Resultatet visas direkt i konsolfönstret

Data som visas:
• name  
• description  
• html_url  
• homepage  
• watchers  
• pushed_at
